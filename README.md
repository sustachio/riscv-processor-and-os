very much in progress thing here. this is a risc-v (RV32I_Zicsr for now) fpga processor implementing the basic M-mode (and U-mode) privliged ISA spec targeting the old DE1 fpga boards. code for the processor is done in verilog and can be foudn in `src/`. the processor can run code generated by gcc and you can find some example programs in `c_coding/fullprograms`. i'll do a better write up once its at a better state but right now it has vga out and ps/2 keyboard in and can run conways game of life. i'm currently working on a minimal OS which will eventually be able to load programs off of an SD card (currently everything must be put onto the flash chip).
